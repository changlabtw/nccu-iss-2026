# NCCU ISS 2026 — Digital Communication and Emerging Technologies

Course website for the NCCU International Summer School 2026 course **S3: Digital Communication and Emerging Technologies**.

## Publish it (GitHub Pages, no command line)

1. On [github.com](https://github.com), click **New** to create a repository. Name it something like `nccu-iss-2026` and make it **Public**. Do **not** add a README (this folder already has one).
2. On the new empty repo page, click **uploading an existing file**.
3. Drag **all the files in this folder** (index.html, style.css, the three `.html` pages, the three `.docx` files, `.nojekyll`, and this README) into the upload area, then **Commit changes**.
4. Go to **Settings → Pages**. Under *Build and deployment*, set **Source: Deploy from a branch**, **Branch: `main` / `(root)`**, and **Save**.
5. Wait about a minute, then reload. Your site will be live at `https://<your-username>.github.io/nccu-iss-2026/`. Share that link with students.

## What's in here

| File | Purpose |
|---|---|
| `index.html` | Landing page: schedule, readings, grading, document links |
| `syllabus.html` / `.docx` | Full syllabus |
| `lab-guide.html` / `.docx` | In-class lab guide (run-of-show + worksheets) |
| `supplement.html` / `.docx` | Learning objectives + NotebookLM handout |
| `style.css` | Shared styling |
| `.nojekyll` | Tells GitHub Pages to serve the HTML as-is (no build step) |

To edit content later, change the source Markdown/HTML and re-upload, or edit files directly on GitHub.
